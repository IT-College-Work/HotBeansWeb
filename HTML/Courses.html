Courses

To help you get started on your Web Development Journey



Codecademy

Codecademy offers many courses which are designed for you to increase your skills in programming and markup languages. They also give you some work examples that you can put on your portfolio

Go to Codecademy

w3schools

w3schools is a website dedicated to giving you information on all of the different parts of a language. It contains information about all of the HTML and CSS tags aswell as having resources on JavaScript

Go to w3schools

Mozilla

Mozilla, the creators of Firefox, have a course tailored towards learning Web-Development. It covers a wide range of topics from the basics to server-side website programming

Go to Mozilla

UDemy

UDemy has a great "Web Developer Bootcamp" course. This is a good course for anyone that wants to start from the basics. You will learn how to create good looking and responsive website layouts among other highly useful skills

Go to UDemy

Google

Google has created the Web Fundamentals website with assets on various web-development topics. These topics are intended for both beginners and experienced developers alike

Go to Google Web Fundamentals

FreeCodeCamp

FreeCodeCamp allows users to learn how to program or code for free. It is designed to teach you everything you need to know about the subject and gives you all the valuable skills needed.

Go to FreeCodeCamp
